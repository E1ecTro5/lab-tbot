# Лабораторная работа №3
### Название проекта.
Телеграм-бот. Ничего особенного.  
### Описание.
Простой бот для показа погоды через отправку запросов. P.S.: главное сам факт возможности деплоя, а не приложение.
### Стек технологий
+ Python 3.13
+ dotenv==0.9.9
+ pyTelegramBotAPI==4.33.0
+ requests==2.33.1
### Локальный запуск
Склонируйте репозиторий себе. Добавьте в папку файл .env и пропишите ключи тг-бота и openWeatherAPI (свои давать не буду). Делать в таком формате:
```
BOT_API_TOKEN='{key}'
WEATHER_TOKEN='{key}'
```
### Деплой
Деплой был произведён на платформе Railway, с функцией деплоя GitHub-репозитория. Единственное, что стоит "подкрутить" - это локальные переменные (т.е. те самые ключи/токены).
